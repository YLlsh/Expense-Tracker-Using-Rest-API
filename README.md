## Description
Expense Tracker is a web application built using Django and Django Rest Framework (DRF) that helps users track and manage their daily expenses. The project uses PostgreSQL as the database and provides secure authentication using JWT. The frontend is developed using HTML and CSS.

## Features
- User Signup
- User Login with JWT Authentication
- Access and Refresh Token system
- Secure API endpoints
- Add, update, delete, and view expenses  (pending)
- Category-based expense
- Filter with date range
- RESTful APIs using Django Rest Framework

## Tech Stack
- Django
- Django Rest Framework (DRF)
- PostgreSQL
- HTML
- CSS
- JavaScript
- JWT Authentication (Simple JWT)

## Authentication
- JSON Web Token (JWT) based authentication
- Access Token for protected API requests
- Refresh Token to generate a new access token
- Authorization header used for secured endpoints

## Frontend
- HTML used for structure
- CSS used for styling
- JavaScript used for API integration and token handling

## Future Enhancements
- Export expenses to CSV or PDF
- Password reset and email verification
